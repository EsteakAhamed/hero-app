# App Name

Hero App

---

# Description

Hero App is a small single-page React application built with Vite. It demonstrates listing apps, viewing app details, and basic UI interactions. The project uses modern frontend tooling for fast development and styling.

---

# Technologies

- React
- Vite
- Tailwind CSS
- DaisyUI
- React Router (react-router / react-router-dom)
- Recharts
- React Toastify
- ESLint
