Hero App is a sophisticated single-page React application built with Vite that provides users with:
- **App Discovery**: Browse through a comprehensive list of mobile applications
- **Detailed Information**: View in-depth details about each application
- **Interactive UI**: Smooth navigation and user-friendly interface
- **Responsive Design**: Fully responsive design that works on all devices
- **Data Visualization**: Visual representations of app statistics and information

This project demonstrates modern React patterns, component-based architecture, and best practices in frontend development using cutting-edge tools and frameworks.

## ğŸš€ Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn package manager

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/EsteakAhamed/hero-app.git
   cd hero-app
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```
   The app will be available at `http://localhost:5173`

### Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint to check code quality
